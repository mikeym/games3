What's a Grid System?

  • Typographic grid systems are a key structure of graphic design layout
  • Columns (content) and Gutters (spaces between content)
  • Number of columns may change at different viewport widths
  • Gridinator
  • 960 Pixel Grid System
  • 1200 Pixel Grid System
  • 1140 Pixel Grid System

What are Responsive Images and Media?

  • Fill available space within the layout grid
  • example 1   example 2   example 3
  • max-width: 100% forces an image to match the width of its container
  • Note difference between CSS width and internal width of a Canvas element
  • IE6, IE7 scaling artifacts etc
<figure style="width: 20%; display: inline-block;">
  <img src="toad.jpg" style="max-width: 100%;">
  <p>20%</p>
</figure>

What are Media Queries?

  • CSS2 stylesheet spec
  • Media types are useful for screen and print stylesheets, not for handhelds
  • Media queries can reliably inspect physical characteristics and features
  • See MDN etc for more info
  • Interactions with viewport meta tag affect device scaling
@media screen and (min-width: 1024px) {
  body { margin: 1em 4em; }
}
@media screen and (min-width: 400px) {
  body { margin: 0 1em; }
}
